Book Store App

Description

A versatile book store app built with Flutter, Dart, and Firebase. This app allows users to browse a wide range of available books, purchase them, and have them shipped directly to their address. Users can also create a wishlist of books, which will notify them when those books become available. Admins have the ability to add or delete books based on availability. The app also includes features for managing user accounts and tracking orders.

Tech Stack

  • Frontend: Flutter, Dart
  • Backend: Firebase
  • Version Control: GitHub

GitHub Repository: View Code on GitHub

Challenges Faced

  • Inventory Management: Implementing a robust system to manage book availability and updates in real-time was challenging. Ensuring that the app reflects accurate inventory levels and handles out-of-stock scenarios required careful planning and integration with Firebase.
  • Wishlist Notifications: Developing a notification system to alert users when books from their wishlist become available involved integrating Firebase Cloud Messaging and handling various scenarios where users might want to be notified.
  • UI/UX Design: Creating an intuitive and visually appealing interface that enhances the shopping experience for users and provides easy management options for admins required extensive design and usability testing.
  • Order Processing: Implementing a secure and efficient order processing system to handle transactions, manage shipping, and track order status presented technical and logistical challenges, particularly with integrating payment gateways and ensuring data security.
