Gym/Fitness App

Description

A comprehensive gym management app built with Flutter, Dart, and Firebase. This app allows users to subscribe or renew their gym membership, monitor their attendance, layout their workout regimen, sign up for available classes, book trainers, and receive notification reminders. Users can also update their profiles. The admin section provides tools to manage user data, add updates, and monitor daily attendance.

Tech Stack

  • Frontend: Flutter, Dart
  • Backend: Firebase
  • Version Control: GitHub

GitHub Repository: View Code on GitHub

Challenges Faced

  • State Management: Managing the state of multiple features such as workout plans, class bookings, and notifications was challenging. I had to carefully structure the state management to ensure the app remained responsive and efficient as the user interacted with various parts of the app.
  • Firebase Integration: Integrating Firebase to handle real-time data updates, authentication, and notification services required a deep understanding of both Firebase and how it interacts with Flutter. Ensuring secure data handling, especially for user profiles and payment information, was a critical part of the development process.
  • User Experience Design: Designing an intuitive user interface that caters to both gym members and administrators required several iterations. Balancing functionality with ease of use, especially for non-tech-savvy users, was a key focus.
  • Scalability: Building the app to handle a growing number of users and data without compromising performance posed challenges in database structuring and API optimization. Implementing efficient querying and data retrieval processes was essential to maintaining app speed and reliability.
